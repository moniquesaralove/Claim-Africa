CLAIM AFRICA — QUICK VERCEL DEPLOYMENT

You're already on Vercel. Here's the fastest path:

STEP 1: Extract the files
- Download claim-africa-vercel.tar.gz
- Extract it: tar -xzf claim-africa-vercel.tar.gz
- Or just download the individual files from the outputs folder

STEP 2: Push to GitHub
You need a GitHub repo. Do this once:
  git init
  git add .
  git commit -m "Initial commit"
  git branch -M main
  git remote add origin https://github.com/YOUR_USERNAME/claim-africa.git
  git push -u origin main

STEP 3: Import to Vercel
1. Go to vercel.com and log in
2. Click "Add New..." → "Project"
3. Select your GitHub repo (claim-africa)
4. Click "Import"
5. Click "Deploy"

DONE! Wait ~1 minute and you'll get a URL like:
  https://claim-africa-xyz.vercel.app

STEP 4: Play
- Open that URL
- Click "Copy invite link"
- Send the link to your friend
- Both tap the map to play

NEED HELP?

Can't deploy:
- Make sure you have a GitHub account (free)
- Make sure Vercel is linked to GitHub
- Check your repo is public or private is OK

Game not syncing:
- Both players must use the same URL (with room code)
- Refresh the page if stuck
- URL format: https://claim-africa-xyz.vercel.app#roomcode
